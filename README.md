# Darwin Coffee

Проект для автоматизации аналитики кофейни: Telegram-бот, отчёты по выручке, продажам, прибыли и расходам.

## Что уже есть

- Базовая структура проекта
- `package.json` для Node.js + TypeScript
- `tsconfig.json`
- `src/index.ts` — точка входа

## Дальнейшие шаги

1. Выбрать стек: Node.js / Python
2. Добавить интеграцию с Telegram
3. Настроить базу данных (PostgreSQL)
4. Реализовать загрузку данных из кассы или Google Sheets

## Запуск

```bash
npm install
npm run dev
```
